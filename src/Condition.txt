Требования:
Для реализации необходимо:
+	Создать класс “Battleship”
+	Создать переменные для хранения имен игроков. Ввод текста происходит через консоль.
?   При необходимости создайте имена-заглушки
+	Реализовать метод для создания игрового поля с использованием циклов
•	Реализовать расположения кораблей. 
•	Обратите внимание на тонкости работы с консолью при отображении игровых объектов. 
•	Создать геймплей. 
•	Обработать исключения, которые могут возникать.
+	Исходный код должен быть размещен на github или другом подобном сервисе
